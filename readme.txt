== Description ==

Easily integrate your AdWork Media Content Lockers & Link Lockers into your WordPress site.

This plugin will allow you to easily setup AdWork Media Content Lockers & Link Lockers on pages and posts of your choice. An [AdWork Media](http://www.adworkmedia.com/ "Get an AdWork Media account") account is required.

Overview:

*   Easily filter pages or posts for the locker tools to "always display" or "never display".
*   No need to edit the header.php file or direct template files.
*	Supports multiple Content Locker & Link Locker profiles.
*   Supports auto Link Locking so integration is quick and easy.

== Installation ==

1. Upload the `adwork-media-ez-content-locker` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create a new Content Locker profile and enter your AdWork Media Content Locker (Gateway) JavaScript code into the appropriate box.
4. Alternatively, you could create a new Link Locker profile and enter your Link Locker API Key to quickly setup auto Link Locking with your site.
